# 🤖 n8n AI Automation Project

This repository contains an AI-powered automation workflow built with **n8n**. It demonstrates how to interact with a data store to query and process information efficiently.

## ✨ Features
* **Manual Trigger:** Allows easy testing and execution.
* **Data Store Integration:** Queries "All People" data from a customer database.
* **Automated Processing:** Handles data flows seamlessly using n8n nodes.

## 🛠️ How to Use
1. Install **n8n** on your machine or use the cloud version.
2. Download the `data store query part.json` file from this repo.
3. Import the file into your n8n canvas.
4. Configure your credentials (if any) and click **Execute Workflow**.

## 🚀 Technologies Used
* [n8n.io](https://n8n.io/) - Workflow Automation Tool
* JSON - Data structure for the workflow
